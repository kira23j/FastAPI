# Hello Flask

A minimal first Flask app that displays "Hello, Flask!" on the homepage.

## Usage

1. Install Flask:

    ```bash
    pip install Flask
    ```

2. Run the app:

    ```bash
    python app.py
    ```

3. Open your browser and go to [http://127.0.0.1:5000/](http://127.0.0.1:5000/) to see the message.
